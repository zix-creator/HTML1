เว็บไซต์แนะนำคำสั่งHTML

นางสาวเขมิกา กาฬวงศ์ 704

นายธนกร สุขสวัสดิ์ 711

1. <p> Paragraph

คำสั่ง <p> ใช้สำหรับแสดงข้อความเป็นย่อหน้า

2. <h1> - <h2> Heading

คำสั่ง Heading ใช้แสดงหัวข้อ โดย <h1> ใหญ่ที่สุด

3. <b> Bold

คำสั่ง <b> ใช้ทำให้ข้อความเป็นตัวหนา

4. <i> Italic

คำสั่ง <i> ใช้ทำให้ข้อความเป็นตัวเอียง

5. <br> Line Break

คำสั่ง <br> ใช้ขึ้นบรรทัดใหม่

6. <hr> Horizontal Rule

คำสั่ง <hr> ใช้สร้างเส้นคั่นแนวนอน

7. <ul> Unordered List

ใช้ <ul> และ <li> สร้างรายการแบบจุด

8. <ol> Ordered List

คำสั่ง <ol> ใช้สร้างรายการแบบตัวเลข

9. <a> Link

คำสั่ง <a> ใช้สร้างลิงก์ไปยังเว็บไซต์อื่น

10. <img> Image

คำสั่ง <img> ใช้แสดงรูปภาพ

11. <table> Table

คำสั่ง <table> ใช้สร้างตารางข้อมูล

12. <button> Button

คำสั่ง <button> ใช้สร้างปุ่ม

13. <input> Input

คำสั่ง <input> ใช้รับข้อมูลจากผู้ใช้

14. <form> Form

คำสั่ง <form> ใช้สร้างแบบฟอร์ม

15. <div> Division

คำสั่ง <div> ใช้จัดกลุ่มและจัดรูปแบบเนื้อหา
